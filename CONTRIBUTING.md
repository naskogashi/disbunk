# Contributing to Disbunk.org

Thank you for your interest in contributing to Disbunk.org — an open-source platform for rapid, credible, coordinated response to digital misinformation.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/your-username/disbunk.git`
3. Install dependencies: `npm install`
4. Start dev server: `npm run dev`

## Development

- **Stack**: React + TypeScript + Tailwind CSS + shadcn/ui
- **Backend**: Supabase (auth, database, storage, edge functions)
- **i18n**: Albanian (sq) + English (en) via react-i18next

## Code Style

- Use TypeScript strict mode
- Follow existing component patterns
- Use semantic design tokens from `index.css` — never hardcode colors
- Write meaningful commit messages

## Pull Requests

1. Create a feature branch: `git checkout -b feature/your-feature`
2. Make your changes
3. Run tests: `npm test`
4. Push and open a PR against `main`

## Reporting Issues

Use GitHub Issues. Include:
- Steps to reproduce
- Expected vs actual behavior
- Browser and OS info

## Code of Conduct

Be respectful, constructive, and inclusive. We're fighting misinformation together.
